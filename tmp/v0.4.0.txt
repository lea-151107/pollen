pollen v0.4.0

Usability release focused on collection management, response exploration,
and scripted startup.

Highlights:
- Filter highlighting: matching text in History/Collections filter is
  shown bold+underline so matches are immediately visible
- Collection editing: rename an entry with e, or load an entry then
  press Ctrl+B to update it in-place (Enter) or save as new (n)
- Response diff (D): character-level diff of the previous response body
  vs the current one; green = added, red strikethrough = removed;
  stays active across successive sends
- CLI flags: --env <name>, --config <dir>, --collection <name>

Fixed: diff toggle-off now restores a locked jq filter view; collection
name color preserved correctly after a filter highlight; save-as-new
clears the update prompt for subsequent Ctrl+B presses.

See CHANGELOG.md for the full list.
